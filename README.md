# detection-of-adulteration-in-coffee-using-HSI-and-ML
This was a work done as part of the Intensive Research Training Program (IRTP) at GUGC in 2025. Two research programs were taken in succession, and contents of both were integrated into this project.
First was the AI Winter School Beginners Program, where selected students were given opportunities to get an introduction to the field of machine learning (ML).
Second was the detection of coffee adulteration using hyperspectral imaging (HSI) and ML. It involved lab work preparing coffee samples, and using HSI camera for data collection.
As a result, the final output is 'scuffed', but it served well as its purpose of being my first ML project.
